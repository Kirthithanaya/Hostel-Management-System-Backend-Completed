# Hostel Management System - Backend

This is the backend for the Hostel Management System, built with Node.js, Express, and MongoDB.

## Features

- Resident Management
- Room Allocation
- Maintenance Requests
- Billing & Payments
- Financial Reporting
- User Roles & Permissions
